# ¿Que es este repositorio? ¿Que finalidad tiene?.

¡Hola, curioso lector! Mi nombre es Elizabeth Ochoa, amante de las tecnologías de la información y desarrolladora frontend junior.

Este repositorio en GitHub es un nuevo proyecto que tengo en mente para retomar las riendas de mi autoaprendizaje y fortalecer continuamente mis conocimientos en desarrollo web. Cada día es una nueva oportunidad para aprender algo nuevo y también para recordar buenas prácticas de programación; por eso, he tomado la decisión de desarrollar un espacio en el que pueda estar actualizando mi perfil profesional, llevando una bitácora donde pueda almacenar mis proyectos y mostrarlos al mundo para así demostrar mis capacidades como programadora.

La finalidad de este proyecto es crear un espacio de aprendizaje constante y tener al día mis conocimientos, para así llegar a lugares donde mis talentos y habilidades marquen, día a día, un avance progresivo. Asimismo, busco tener un espacio donde pueda experimentar, crear y desarrollar nuevos proyectos como este e incluso cada vez más grandes.

Espero que este proyecto marque un antes y un después en mi carrera profesional y que algún día pueda mirar atrás y darme cuenta de lo lejos que he llegado.

Un enorme saludo y hasta pronto.
